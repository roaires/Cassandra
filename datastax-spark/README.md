
Exemplo de implementação de Apache Cassandra ( Datastax ) com integração com Spark.

Objetivos:
- Criação de KEYSPACE
- Criação de tabela
- Manipulação de dados no Cassandra
- Criação de um RDD no Spark acessando Cassandra
- Map e Reduce no Spark
- Transformação de RDD em tabela no Cassandra
